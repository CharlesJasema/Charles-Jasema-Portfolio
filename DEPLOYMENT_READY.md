## 🌐 Deployment Instructions

### Quick Deploy to Vercel (5 minutes)

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Production ready - Portfolio system complete"
   git push origin main
   ```

2. **Deploy to Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Import your GitHub repository
   - Add environment variables from `.env.local`
   - Deploy (automatic)

3. **Configure Services**
   - Set up Google Analytics 4 with your measurement ID
   - Configure Tawk.to live chat with your property ID
   - Test all functionality

### Environment Variables for Production
```env

NEXT_PUBLIC_SANITY_DATASET=production
NEXT_PUBLIC_SANITY_API_VERSION=2024-01-01
SANITY_API_TOKEN=your_production_token

NEXT_PUBLIC_GA_MEASUREMENT_ID=G-XXXXXXXXXX
NEXT_PUBLIC_TAWK_TO_PROPERTY_ID=your_tawk_property_id
```
