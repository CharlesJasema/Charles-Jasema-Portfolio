## 🆘 Troubleshooting

### If Lyrics Don't Show
1. Check browser console for errors
2. Verify lyrics.ts syntax is correct
3. Clear browser cache
4. Refresh the page

### If Download Doesn't Work
1. Check browser allows downloads
2. Verify lyrics have content
3. Try different browser
4. Check file permissions

### If Search Doesn't Work
1. Verify search query is entered
2. Check lyrics data is loaded
3. Try different search terms
4. Refresh the page

### If NEW Badges Don't Show
1. Verify `isNew: true` in config
2. Check CSS animation is working
3. Clear browser cache
4. Verify Tailwind classes
