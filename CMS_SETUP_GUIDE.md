# Content Management System Setup Guide

## Overview
Your portfolio uses Sanity CMS for content management. This guide will help you set up and manage your content effectively.

## Accessing the Admin Panel

### Local Development
```
URL: http://localhost:3001/admin
```

### Production (after deployment)
```
URL: https://your-domain.com/admin
```

## Sanity Studio Features

### 1. Content Types Available
- **Songs**: Manage your music catalog
- **Music Videos**: Upload and organize video content
- **Lyrics**: Store song lyrics with metadata
- **Portfolio Projects**: Showcase your work
- **Blog Posts**: Write and publish articles

### 2. Content Organization
The admin panel is organized into sections:
- 🎵 **Music Content**: Songs, videos, and lyrics
- 💼 **Portfolio**: Professional projects
- 📝 **Blog**: Articles and posts
- ⭐ **Featured Content**: Highlighted items
- 🆕 **Recent Content**: Latest additions

## Adding Content

### Adding a New Song
1. Navigate to "Music Content" → "Songs"
2. Click "Create new Song"
3. Fill in required fields:
   - Title
   - Artist (your name)
   - Release Date
   - Genre
   - Audio File (upload MP3/WAV)
   - Cover Image
   - Description
4. Optional fields:
   - Lyrics
   - Collaborators
   - Recording Details
   - Spotify/Apple Music links
5. Toggle "Featured" to highlight on homepage
6. Click "Publish"

### Adding a Portfolio Project
1. Navigate to "Portfolio"
2. Click "Create new Project"
3. Fill in project details:
   - Title
   - Category (Software, Design, Music, Video)
   - Description
   - Technologies Used
   - Project Images
   - Live URL
   - GitHub Repository
   - Client Information
4. Add project challenges and solutions
5. Toggle "Featured" for homepage display
6. Click "Publish"

### Adding Blog Posts
1. Navigate to "Blog"
2. Click "Create new Blog Post"
3. Complete the form:
   - Title
   - Slug (URL-friendly version)
   - Content (rich text editor)
   - Featured Image
   - Tags
   - Meta Description
   - Publish Date
4. Preview before publishing
5. Click "Publish"

## Content Best Practices

### Image Guidelines
- **Minimum Resolution**: 1200px width for featured images
- **Aspect Ratios**: 16:9 for videos, 1:1 for album covers
- **File Formats**: JPG for photos, PNG for graphics, WebP preferred
- **File Size**: Keep under 2MB for web optimization
- **Alt Text**: Always add descriptive alt text for accessibility

### SEO Optimization
- Write compelling meta descriptions (150-160 characters)
- Use descriptive, keyword-rich titles
- Add relevant tags to blog posts
- Include internal links between related content
- Optimize images with descriptive filenames

### Content Structure
- Use clear, descriptive headings
- Break up long text with subheadings
- Include relevant images and media
- Add calls-to-action where appropriate
- Keep paragraphs concise and scannable

## Media Management

### Audio Files
- **Supported Formats**: MP3, WAV, FLAC
- **Quality**: 320kbps MP3 or higher
- **Metadata**: Ensure proper ID3 tags
- **File Naming**: Use descriptive names

### Video Files
- **Formats**: MP4 (H.264 codec recommended)
- **Resolution**: 1080p minimum, 4K preferred
- **Compression**: Balance quality and file size
- **Thumbnails**: Upload custom thumbnails

### Document Files
- **CVs/Resumes**: PDF format
- **Press Kits**: PDF or ZIP files
- **Sheet Music**: PDF format
